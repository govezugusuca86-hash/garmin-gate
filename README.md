# GarminGate — Интернет-магазин

Статический интернет-магазин спортивной электроники Garmin, Polar, Suunto.

## 🚀 Деплой на GitHub Pages

### Шаг 1 — Создать репозиторий
1. Зайди на [github.com](https://github.com) → **New repository**
2. Название: `garmin-gate` (или любое другое)
3. Поставь галочку **Public**
4. Нажми **Create repository**

### Шаг 2 — Загрузить файлы
**Вариант A — через браузер (проще):**
1. В репозитории нажми **uploading an existing file**
2. Перетащи все файлы и папки проекта
3. Нажми **Commit changes**

**Вариант B — через Git (командная строка):**
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/ВАШ_ЛОГИН/garmin-gate.git
git push -u origin main
```

### Шаг 3 — Включить GitHub Pages
1. В репозитории → **Settings** → **Pages** (в левом меню)
2. Source: **Deploy from a branch**
3. Branch: **main** → **/ (root)**
4. Нажми **Save**

### Шаг 4 — Сайт готов! 🎉
Через 1–2 минуты сайт будет доступен по адресу:
```
https://ВАШ_ЛОГИН.github.io/garmin-gate/
```

## 📁 Структура файлов

```
garmin-gate/
├── index.html          # Главная страница (каталог)
├── css/
│   └── style.css       # Все стили
├── js/
│   ├── app.js          # Логика (корзина, фильтры, и т.д.)
│   └── products.js     # Данные товаров из XML (461 товар)
└── pages/
    ├── product.html    # Страница товара
    ├── cart.html       # Корзина
    ├── checkout.html   # Оформление заказа
    ├── success.html    # Страница успешного заказа
    └── favorites.html  # Избранное
```

## ✨ Функционал

- 📦 **461 товар** из XML, разбитых по 10 категориям
- 🔍 **Поиск** по названию и бренду
- 🗂️ **Фильтры**: по категории, бренду, цене, скидкам
- 🛒 **Корзина** с управлением количеством
- ❤️ **Избранное**
- 📱 **Адаптивный дизайн** (мобильный, планшет, десктоп)
- ✅ **Оформление заказа** с формой
- 📄 **Страница товара** с описанием и вкладками

## 🔧 Обновление товаров

Чтобы обновить товары, замени файл `js/products.js` —  
данные хранятся в переменной `PRODUCTS_DATA`.
